# 📚 LibraryLaravel

Selamat datang di proyek **LibraryLaravel**! Ini adalah aplikasi manajemen perpustakaan digital berbasis web yang dibangun menggunakan **Laravel 13** dan database **MySQL**.

---

## 📦 Link Download Repositori LibraryLaravel
Klik link di bawah ini untuk mendownload repository secara cepat:

* [Download (ZIP)](https://github.com/AkmalFauzan1945/LibraryLaravel/archive/refs/heads/main.zip)

---
  
## 🚀 Fitur Utama
Aplikasi ini memiliki beberapa modul utama yang sudah selesai dikembangkan:
* **Manajemen Kategori:** Melihat daftar kategori apa saja yang tersedia.
* **Manajemen Buku:** Mengelola data buku seperti Judul, Kategori, Media, Pengarang, Penerbit, Tahun dan Status.

---

## ⚙️ Teknologi yang Digunakan
* **Framework:** Laravel 13
* **Bahasa Pemrograman:** PHP & JavaScript
* **Database:** MySQL (XAMPP)
